# Tonal Recall

A musical memory matching game built with vanilla HTML, CSS, and JavaScript.

## Features

- **Musical Memory Game**: Match pairs of cards by their musical tones
- **Multiple Grid Sizes**: Choose from 4x4 (Easy), 4x6 (Medium), or 6x6 (Hard)
- **Customizable Scales**: Select from various musical scales (Major, Minor, Dorian, Phrygian, Lydian, Mixolydian, Locrian)
- **Root Note Selection**: Choose any root note from C to B
- **Audio Feedback**: Each card plays a unique tone from the selected scale
- **Victory Jingle**: Celebratory melody when you win
- **Clean UI**: Settings panel before gameplay, clean grid during play

## How to Play

1. Open `index.html` in a web browser
2. Select your preferred root note, scale quality, and grid size
3. Click the "Play" button to start
4. Click cards to reveal their tones
5. Match all pairs to win!

## Technologies

- HTML5
- CSS3 (with glassmorphism effects)
- Vanilla JavaScript
- Web Audio API

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and animations
- `script.js` - Game logic and audio generation
